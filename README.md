

<h3>Basic project done using TypeScript Tailwind and React. </h2>


To clone the repo :

```
git clone https://github.com/Zrital3265/BasicPasswordGenerator 
```

Navigate to the root directory and type this in your terminal :

```
npm install
```
then use :

```
npm run dev
```
